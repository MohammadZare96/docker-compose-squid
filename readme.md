# Squid Proxy run as docker (docker-compose)





## Run Squid Proxy

1. clone this project 
    ```cmd
    git clone https://github.com/beigi-reza/squid-docker-compse.git
    ```
2. cd squid-docker-compse
3. Run compse File
   ```
   docker compse up 
   ```


## Use 

set http proxy to 127.0.0.1:3228 in firefox
